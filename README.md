 Workshop Overview

This repository highlights my learnings and mini-projects developed during an IoT Hands-On Hardware Workshop, where I worked with multiple microcontrollers, sensors, and IoT modules to design, program, and deploy real-time IoT systems.

The training covered everything from basic circuit designing to cloud-based automation and data visualization using Blynk, ThingSpeak, and Arduino IDE platforms.
